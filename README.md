For app init.

cd my-app

npm install react-scripts --save

npm install @fortawesome/fontawesome-svg-core @fortawesome/free-brands-svg-icons @fortawesome/free-regular-svg-icons @fortawesome/free-solid-svg-icons @fortawesome/react-fontawesome

cd ..

cd back-end

npm install cors

npm install dotenv

npm install express

npm install mysql2

npm install sequelize

npm install axios

npm install nodemon

React server can be started from my-app using npm start, Back end can be started from back-end using node index.js
